## 워킹 폴더 생성

1. $ npm install -g express-generator
2. $ express {프로젝트 명}
3. $ cd 해당 폴더 이동
4. $ npm install  // 이 경우 package.json 에 있는 것들을 install해줌
5. $ npm install ejs
6. $ node server.js 로 실행
